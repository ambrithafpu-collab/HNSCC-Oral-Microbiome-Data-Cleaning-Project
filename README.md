# HNSCC Oral Microbiome Data Cleaning

## 🔬 Objective
To clean and preprocess oral microbiome dataset related to Head and Neck Squamous Cell Carcinoma (HNSCC)

## 🧬 Background
The oral microbiome plays an important role in cancer development and progression. This project focuses on preparing microbiome data for analysis.

## 🛠️ Tools Used
- Python
- Pandas

## ⚙️ Process
- Loaded raw dataset
- Removed missing values
- Cleaned and formatted data
- Prepared dataset for further analysis

## 📊 Results
- Clean dataset ready for microbiome analysis
- Improved data quality for downstream bioinformatics tasks

## 🚀 Future Work
- Data visualization
- Microbial abundance analysis
- Statistical analysis
